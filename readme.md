#### Sección A
Sin hacer uso de librerías o funciones nativas. Hacer un programa que tomando una palabra o párrafo:
  - Contar cuántas vocales hay.
  - Reemplazar las vocales por la vocal consecutiva (a->e).

```
python vocals.py
```
